## Description

¿Cansado de que el código base de tu aplicación Android crezca desmesuradamente? ¿Pasas más tiempo esperando los builds que agregando código? Si este es tu caso entonces esta multi-modularizar tu app puede ser la solución :)

La técnica de multi-módulos es una pieza clave para gestionar un código base creciendo de manera compleja pero no es la única ventaja es por eso que en esta charla hablaremos de:

* Aspectos a tomar en cuenta para modularizar tu aplicación.
* Ventajas y desventajas de la modularización.
* Los tipos de módulos que pueden surgir.
* Cómo crear un módulo (implementado en código).
* Cómo navegar/pasar información entre diferentes módulos
* Experiencias buenas/fallidas implementando modularización
