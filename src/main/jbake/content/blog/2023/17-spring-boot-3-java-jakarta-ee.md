## Description

En esta charla abordaré acerca de la transición de Java EE a Jakarta EE, sus motivaciones, beneficios y cómo ha impactado de manera general al ecosistema de Spring. También se abarcarán temas relacionados con el desarrollo de Spring Framework 6 y Spring Boot 3, requerimientos en cuanto a la especificaciones de Jakarta EE y JDK soportado y de lo qué se va a desprender Spring en el futuro. Así también , se hablará un poco en cuanto al tiempo de soporte de las actuales versiones de Spring Framework 5 y Spring Boot 2. También abordaremos los puntos que debemos tomar en cuenta al momento de migrar una aplicación de Spring Boot 2 a Spring Boot 3. Finalizaremos con una vista en perspectiva acerca de los factores qué deberíamos tomar en cuenta a elegir una versión y distribución de Java que mejor se adapte a nuestros desarrollos con Spring Boot 3
